# Retro UI Component Library

A nostalgic UI component library styled like early 2000s interfaces.  
Built with vanilla HTML, CSS, and JavaScript – no frameworks, no external dependencies.  
Includes all essential form elements and controls with a vintage look and feel.

**Live Demo:** [https://old-uikit.netlify.app/](https://old-uikit.netlify.app/)

---

## Features

- **Retro aesthetic** – beveled borders, pixel‑perfect shadows, muted colour palette, and CRT‑style typography.
- **Fully interactive UI components**:
  - **Buttons** – with hover, active, and disabled states (including raised 3D effect).
  - **Input fields** – text, password, email, with focus and validation styling.
  - **Radio buttons & Checkboxes** – custom‑styled with old‑school check marks and dots.
  - **Select dropdowns** – styled to match the retro theme, with custom arrow.
  - **Tables** – striped rows, bordered cells, and hover highlighting.
- **Pure CSS & vanilla JS** – no libraries, lightweight and fast.
- **Keyboard accessible** – all interactive elements are fully usable via keyboard.

---

## Built with

- **HTML5** – semantic markup and accessible forms.
- **CSS3** – Flexbox, Grid, custom properties, `@keyframes`, `filter`, `box-shadow`, `border-style` (for raised/inset effects).
- **Vanilla JavaScript (ES6)** – event handling, dynamic state management (e.g., disabling buttons, toggling checkboxes), and simple form validation.

## Getting Started

To run this project locally:

1. Clone the repo:
   ```bash
   git clone https://github.com/Florence100/Old_UI/tree/master/Championship-UI.git
   ```
2. Open `index.html` in your browser.
